## Basic Usage

If you're looking to begin developing on the website, you can install the dependencies directly via [Yarn](https://yarnpkg.com/):

```bash
yarn
```

Or if you prefer [NPM](https://npmjs.com):

```bash
npm install --save @outlet-delivery/signal
```
